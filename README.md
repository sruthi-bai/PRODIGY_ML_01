# PRODIGY_ML_01

## House Price Prediction using Linear Regression

This project implements a Linear Regression model to predict house prices based on:
- Square Footage
- Number of Bedrooms
- Number of Bathrooms

### Technologies Used
- Python
- Pandas
- Scikit-learn

### Description
The model is trained using a CSV dataset and evaluated using a train-test split approach. 
Users can input house details to predict the estimated price.

### Outcome
Successfully predicted house prices using Linear Regression.
